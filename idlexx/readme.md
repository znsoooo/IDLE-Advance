- __init__.py
运行run.py获得一个加载所有插件的IDLE-Advance的示例文件
运行__init__.py获得一个打开自身脚本并加载所有插件的editor的例子
分别运行idlexx目录下的扩展文件，可以得到一个打开自身的editor的例子（有的是shell的例子）
如果需要个别扩展不需要加载，将对应的文件从文件夹目录内移除后再次运行即可
